What is CS:Legacy?

CS:Legacy is a classic old-school shooter, built from the ground up on Valve's public 2013 Source Engine SDK.

Using 100% custom game code and game assets (textures, models, animations, UI, etc), and our own major rewrites to the renderer, shaders and various systems.

Key Features:

  • Customised graphics system
  • Physically Based Rendering
  • Completely rewritten reflection system
  • Custom UI and in-house source engine tools

Game Features

Classic Gameplay

Experience the fast-paced, tactical gameplay reminiscent of classic shooters with modern enhancements.

Custom Graphics

Our advanced custom branch features physically based rendering and a rewritten reflection system.

Multiplayer Focus

Compete in team-based matches that reward skill, strategy and coordination.

Release Information

When will CS:Legacy release?

CS:Legacy is planned to release on Steam in 2025 at an Early Access stage, and free for everyone to play.

Which engine is CS:Legacy running on?

CSL is running on its own advanced custom branch of Valve's official Source Engine 2013 public SDK.

Our branch features a customised graphics system, Physically Based Rendering, completely rewritten reflection system, custom UI, custom in-house source engine tools and many more features.
